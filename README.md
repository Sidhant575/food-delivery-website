# Responsive Food Delivery Website

A responsive food delivery website built using **HTML5** and **CSS3**. This project focuses on creating a clean, modern, and responsive user interface using only HTML and CSS.

## Technologies Used

* HTML5
* CSS3

## Features

* Responsive layout
* Navigation bar
* Food menu section
* Contact section
* Clean and modern design
